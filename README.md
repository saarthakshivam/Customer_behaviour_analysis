# 📊 Data Analytics Project

## 🔍 Overview
This project demonstrates an end-to-end data analytics workflow, starting from raw data processing to generating actionable insights. It covers data loading, cleaning, exploratory data analysis (EDA), SQL-based querying, dashboard creation, and final reporting.

The objective is to extract meaningful insights from the dataset and present them in a clear, visual, and business-friendly format.

## ⚙️ Project Workflow

### 1. Data Loading
- Imported dataset using Python  
- Checked structure, data types, and summary statistics  

### 2. Exploratory Data Analysis (EDA)
- Analyzed distributions and relationships  
- Visualized key trends using charts  
- Identified missing values and anomalies  

### 3. Data Cleaning
- Handled missing values  
- Removed duplicates  
- Standardized and formatted data  

### 4. SQL Analysis (PostgreSQL)
- Loaded cleaned data into PostgreSQL  
- Performed:
  - Aggregations  
  - Joins and filtering  
  - Insight extraction using SQL queries  

### 5. Dashboard Creation (Power BI)
- Built interactive dashboards  
- Added KPIs, charts, and slicers  
- Enabled user-friendly data exploration  

### 6. Reporting & Presentation
- Summarized findings in a report  
- Created a professional presentation using Gamma
