# Customer_behaviour_analysis
📊 Data Analytics Project
🔍 Overview

This project demonstrates an end-to-end data analytics workflow, starting from raw data processing to generating actionable insights. It covers data loading, cleaning, exploratory data analysis (EDA), SQL-based querying, dashboard creation, and final reporting.

The objective is to extract meaningful insights from the dataset and present them in a clear, visual, and business-friendly format.

📁 Dataset
The dataset contains structured data relevant to analysis (e.g., sales, customers, or operations).
Data was loaded and processed using Python.
Includes features such as numerical, categorical, and time-based variables.
